Library Management System

This is a Basic LMS without any Graphical User Interfaces. Code is written in Python.For storing the all the Data Mysql Database has been used. The Project consist of options to add Student members and books to the Library Database . Students can Issue and Return the book and all the data will be monitored and stored. Students can be fined if the book is not returned in time and the fine amount will be automatically calculated on the basis of book return date. Aside this the Student & Book Details (Details such as Student name, address, class, phone, e-mail & book name, publisher, no. of pages etc.)can be modified.
